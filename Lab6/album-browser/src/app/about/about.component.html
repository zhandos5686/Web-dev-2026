About This Application

This Album Browser is a single-page application built with Angular 18+. It demonstrates the use of:

  • Angular Router with multiple routes and parameters
  • HttpClient for API communication
  • Services for business logic separation
  • Observables and async pipe for reactive programming
  • CRUD operations with a REST API
